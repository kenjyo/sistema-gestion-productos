# Sistema de Gestión de Productos

Aplicación web desarrollada con React (frontend) y Laravel (backend) que permite gestionar productos mediante operaciones CRUD consumiendo una API REST.

## Tecnologías
- React
- Laravel
- MySQL

## Estructura
- app-productos → Frontend
- api-productos → Backend

## Ejecución

### Backend
php artisan serve

### Frontend
npm install
npm run dev
